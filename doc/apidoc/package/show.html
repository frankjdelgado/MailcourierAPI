GET /api/v1/package
Show specified package. If user is a member, it will only show packages related to him.

Supported Formats

json

Examples

{"id":2,"ref_number":"MC-f640505999e450294eb4","description":"package1","weight":21.8673,"height":80.323,"depth":69.3697,"width":7.47692,"value":31.216,"shipping_cost":29.7139,"status":2,"agency_id":8,"date_added":null,"date_arrived":"2014-10-14T12:51:44.000Z","date_delivered":"2014-10-14T12:52:40.000Z","sender_id":1,"receiver_id":2,"created_at":"2014-10-13T08:57:32.000Z","updated_at":"2014-10-14T12:52:40.000Z"}

Params

Param name Description
id
optional

Package database ID


Value: Must be a number.
