GET /api/v1/user/:id
Show specified user

Supported Formats

json

Errors

401 Unauthorized. You can only access your own user details.

Examples

{'id':3,'username':'user2','email':'user2@a.com','role':0}

Params

Param name Description
id
required

User database ID


Value: Must be a number.
