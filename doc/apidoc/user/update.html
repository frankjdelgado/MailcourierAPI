PUT /api/v1/user/:id
Update user attributes

Supported Formats

json

Errors

401 Unauthorized. You can only access with administrators permissions.
400 Bad request. Parameters given are incorrect.

Examples

{"id":7,"username":"testusername3","email":"test2@test.com","role":2}

Params

Param name Description
username
optional

Account username


Value: Must be String
email
optional

Account email


Value: Must be String
agency_id
optional

Agency ID related with user


Value: Must be a number.
password
optional

Account password. Minimun lenght must be 8


Value: Must be String
role
optional

User role inside the application. Can only be specified by an Administrator


Value: Must be one of: 0, 1, 2.
