Resources

Agency

Resource Description
GET /api/v1/agency List agencies
GET /api/v1/agency/:id Show specified agency
POST /api/v1/agency/:id Create an agency
PUT /api/v1/agency/:id Update agency attributes
DELETE /api/v1/agency/:id Delete specified agency

Package

Resource Description
GET /api/v1/package List user packages. If user is an operator, the endpoint will show all packages related to his agency.
GET /api/v1/package Show specified package. If user is a member, it will only show packages related to him.
POST /api/v1/package Create package
PUT /api/v1/package/:id Update package status and dates

Rate

Resource Description
GET /api/v1/rate/calculate Calculate package shipping cost
GET /api/v1/rate List Mailcourier package rates
GET /api/v1/rate/:id Show specified package rate
POST /api/v1/rate Create package rates. Default status 0
PUT /api/v1/rate/:id Update specified package rates attributes
DELETE /api/v1/rate/:id Delete specified package rates

Session

Resource Description
GET /api/v1/session Show user oauth token using basic auth
POST /api/v1/session Generate oauth token using basic auth

User

Resource Description
GET /api/v1/user List users
POST /api/v1/user Create an user
GET /api/v1/user/:id Show specified user
PUT /api/v1/user/:id Update user attributes
